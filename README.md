# Charity and Donation Platform

## 📌 Overview
A full-stack web application that allows users to donate and support people in need. The platform connects donors with donation cases through a simple and user-friendly interface.

## 🚀 Features
- User registration and login system
- Create and manage donation cases
- Secure handling of user data
- Donation tracking system
- Admin dashboard for managing users and content

## 🛠️ Technologies Used
- PHP (Laravel)
- MySQL
- HTML, CSS, JavaScript

## 🎯 Project Goal
The goal of this project is to simplify the donation process and improve transparency by organizing donation cases based on urgency.
